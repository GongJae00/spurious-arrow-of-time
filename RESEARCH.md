# Research Brief

## Core Question

```text
The true task is to infer the hidden cause of an irreversible forward process,
but a stronger non-causal irreversible process can act as a spurious arrow of
time. Which arrow does the model trust?
```

## Motivation

Many physical processes are easy to run forward and hard to invert. Diffusion,
heat flow, spreading smoke, and broken objects all carry a direction of time:
forward evolution tends to erase details about earlier causes.

For machine learning, the question is not only whether a model can detect a
temporal direction. The harder question is whether it can distinguish:

```text
task-causal irreversibility:
  the irreversible process generated by the hidden cause we need to infer

spurious irreversibility:
  another irreversible process that is more visible but not causally relevant
```

## Position Relative To Prior Work

Prior work already studies:

```text
arrow-of-time recognition in video
temporal-order self-supervision
entropy-production estimation from trajectories
spurious correlation and invariant prediction
inverse diffusion and graph source localization
```

The intended gap is the controlled intersection:

```text
irreversible inverse inference + competing non-causal irreversible nuisance
```

See `docs/literature_matrix.md` and `docs/novelty_gap.md` for the current
primary-source audit.

## Current Benchmark Target

The active benchmark is a grid-graph irreversible source inference task.

Core process:

```text
hidden source pattern -> diffusion sequence -> label = source pattern type
```

Nuisance process:

```text
independent moving pulse -> direction correlated with label in train/IID ->
direction reversed or randomized OOD
```

The benchmark must show:

```text
final-frame core evidence is insufficient
full core sequence evidence is sufficient
nuisance direction is a strong train/IID shortcut
nuisance direction fails under OOD shift
static leakage checks remain weak
component visualizations are understandable
```

## Current Evidence State

The benchmark smoke gate and a non-runtime-limited five-seed main neural run
have completed.

Current main result:

```text
core_only_oracle:
  IID and OOD both high

sequence_erm:
  high IID and near-complete OOD collapse under reversed nuisance shift

nuisance_only_oracle:
  same IID/OOD pattern as sequence_erm

counterfactual_invariance:
  reduces OOD gap but sacrifices IID accuracy
```

This supports the benchmark phenomenon but not a positive robustness-method
claim.

## Claim Discipline

Allowed now:

```text
In this controlled benchmark, raw neural sequence predictors can follow a
non-causal irreversible nuisance arrow and fail when that arrow reverses OOD.
```

Allowed with caveat:

```text
The simple counterfactual invariance control is a diagnostic tradeoff, not a
successful robust method, because it lowers IID accuracy substantially.
```

Not allowed:

```text
we solved selective irreversibility
the failure is purely temporal without endpoint/residue contribution
learned latent scores are physical entropy production
neural networks reveal the nature of time
```

```text
Is the benchmark really about a spurious irreversible arrow, or can a model win
from a static endpoint artifact?
```

## High-Risk Neural Result

The main neural result shows:

```text
final_frame_mlp:
  IID = 0.968
  OOD = 0.032
  gap = 0.936
```

This means the mixed final observation contains enough nuisance residue for a
final-frame neural model to exploit the same unstable shortcut.

## Interpretation

This does not erase the spurious-arrow phenomenon, because the nuisance is
generated by a directed irreversible process and the diagnostic static feature
audit is near chance.

It does constrain the paper claim:

```text
Allowed:
  The nuisance arrow leaves visible endpoint/trajectory residue that a neural
  predictor can exploit.

Not allowed:
  The failure is purely temporal or requires long-range sequence reasoning.
```

The benchmark should be described as a strong controlled stress test where a
visible irreversible nuisance process dominates the mixed observation. Any
future stricter benchmark variant should reduce endpoint residue if the paper
wants to isolate temporal reasoning more sharply.

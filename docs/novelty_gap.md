# Novelty Gap

## One-Sentence Claim

```text
Prior work often treats irreversibility as a useful signal to learn, estimate,
or exploit. This project studies when the strongest apparent irreversibility is
the wrong signal for an irreversible inverse task.
```

## What Is New

The novelty is not any single component:

```text
arrow-of-time classification already exists
entropy-production estimation already exists
spurious-correlation/OOD learning already exists
inverse diffusion and source localization already exist
```

The novelty is the controlled intersection:

```text
hidden cause -> irreversible core process -> ambiguous observations -> label

plus

independent nuisance process -> stronger irreversible arrow -> train-only label
correlation -> OOD reversal/removal
```

The central experimental question is:

```text
When a model must infer the cause of an irreversible process, does it trust the
task-causal arrow or the stronger spurious arrow?
```

## Why This Is Not Just Generic Spurious Correlation

Generic spurious-correlation benchmarks often attach an easy non-causal feature
to the label. Here the shortcut must be dynamic:

```text
not color
not brightness
not initial sector
not static texture
not label-coded noise
```

The shortcut is valid only if it is a trajectory-level direction-sensitive
statistic generated by an irreversible process.

## Why This Is Not Just Temporal-Order Learning

Temporal-order learning asks whether a sequence is forward, reversed, shuffled,
or temporally coherent.

This project asks a different question:

```text
The sequence may contain multiple arrows. Which arrow is task-causal?
```

An arrow-of-time classifier can be excellent and still fail the task if it locks
onto the nuisance mechanism.

## Why This Is Not Physical Entropy-Production Estimation

The benchmark can contain irreversible stochastic dynamics, but the project does
not measure heat or exact physical entropy production. Any learned score is a
statistical arrow proxy unless an analytic subsystem is explicitly defined.

## Minimal Evidence Required

The novelty claim is not publishable unless the benchmark gates show:

```text
final-frame source recovery is hard
full-sequence core source recovery is possible
nuisance-only prediction succeeds on IID and fails on OOD
mixed feature probes and actual neural sequence models are reported separately
actual neural sequence ERM follows the nuisance enough to suffer an OOD gap
static leakage probes remain weak
component visualizations make the causal and nuisance mechanisms obvious
```

If these gates fail, the paper should not claim a spurious-arrow benchmark.

## Strong Paper Wording

Use:

```text
We study irreversible inverse inference under spurious arrow shortcuts.
```

Use only after gates pass:

```text
We show that a stronger non-causal irreversible process can dominate learned
sequence predictors even when the task-causal irreversible process is present.
```

Do not use:

```text
we solve the nature of time
neural networks learn thermodynamic causality
our latent score is physical entropy production
```

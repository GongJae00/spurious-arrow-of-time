# Claim Matrix

Use this file as a public claim discipline checklist.

| Claim | Status | Evidence Required |
|---|---|---|
| A dynamic arrow of time can be made spurious in controlled data. | Allowed if diagnostics and evidence pass. | STA-Bench and Ink Advection-Diffusion diagnostics plus evidence audit. |
| OOD failure is tied to nuisance-arrow shift rather than generic corruption. | Conditional. | Closure suite comparing reversed, no-shift, and randomized conditions. |
| SIB/SID improves OOD robustness. | Result-dependent. | `result_interpretation.json` must allow the claim. |
| SID cleanly factorizes reversible, task-irreversible, and nuisance-irreversible factors. | Not assumed. | Factor-role probes must pass; otherwise discuss as diagnostic/negative evidence. |
| Learned latent arrow score is physical entropy production. | Forbidden. | Not measured in this project. |

Default manuscript posture:

```text
controlled diagnostic evidence for a spurious-arrow failure mode
careful method evaluation
no clean factorization claim unless explicitly supported
```
